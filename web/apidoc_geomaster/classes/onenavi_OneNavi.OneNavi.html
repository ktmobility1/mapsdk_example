Class OneNavi

KT 원내비 앱과의 연동 기능을 제공하는 객체입니다.

URL 스키마를 통하여 원내비의 경로탐색 서비스를 호출하는 방식으로 원내비 앱과 연동합니다.

단말에 원내비 앱이 설치되어 있지 않다면 원내비 앱을 설치할 수 있도록 playstore 및 appstore를 호출하고, 원내비 앱이 설치되어 있다면 호출한 출발지/도착지에 맞게 경로탐색 서비스를 호출합니다.

Example

// geomaster import
import geomaster from "kt-map-sdk-geomaster";

let onenavi = new geomaster.OneNavi(); //원내비 생성
onenavi.start({
   start: {
       lat: 37.537229,
       lng: 127.005515,
   },
   end: {
       lat: 37.4979502,
       lng: 127.0276368,
   }
})

  • 원내비 앱을 연동할 수 있는 scheme를 반환합니다
