KT Map SDK Geomaster

KT에서 제공하는 Geomaster 기능을 라이브러리 형태로 사용할 수 있습니다.

제공 기능

[ SEARCH ](Api key 필요)

  • Autocomplete(자동완성)
  • CategoryList(카테고리 리스트)
  • Gas Station(주유소 찾기)
  • Geocode(지오코드/역지오코드)
  • EV Station(전기차 충전소 찾기)
  • Place (POI 장소 찾기)
  • Step by Step (스텝 별 주소 찾기)

[ ROUTE ](Api key 필요)

  • Route (경로 탐색)

[ ONENAVI ]

  • Onenavi (원내비 앱 연동)

사용 방법

  1. 키 발급하기 KT API 사이트 : https://apilink.kt.co.kr/ 에 접속하여 Geomaster Key를 발급합니다.
