class KtMap : MapPluginExtensionsDelegate, CameraListenerDelegate

Android MapSDK에서 상호작용할 일반 클래스입니다. MapView와 관련된 모든 메소드에 대한 진입점을 노출합니다. KtMap 객체를 직접 인스턴스화할 수는 없으며 애플리케이션에 추가한 MapFragment 또는 MapView의 getMapAsync() 메서드에서 객체를 가져와야 합니다.

MapView
Link copied to clipboard
class MapView : FrameLayout

MapView는 삽입 가능한 지도 인터페이스를 제공합니다 이 클래스를 사용하여 지도 정보를 표시하고 애플리케이션에서 지도 콘텐츠를 조작합니다. 주어진 좌표를 중심으로 지도를 배치하고, 표시하려는 영역의 크기를 지정하고, 애플리케이션의 사용 사례에 맞게 지도 기능의 스타일을 지정합니다. 이 뷰를 추가하고 getMapAsync(OnMapReadyCallback)를 호출해 콜백을 등록하면 KtMap 인스턴스를 얻을 수 있습니다.
